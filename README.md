# HNC Contact Management Frontend

React + Vite + TypeScript frontend for the HNC Contacts microservice.

## Quick Start

```bash
pnpm install
pnpm dev
```

## Tech Stack

- React 19
- Vite 6
- TypeScript 5.7
- TanStack Router & Query
- Tailwind CSS

## API

Backend: `https://hncms-contacts.scarif-0.duckdns.org`

## Feature

Part of F10 Contacts - see `_bmad-output/project/features/F10-contacts.md` for specs.
